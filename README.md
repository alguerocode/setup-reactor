## how to setup reactor typescript
setup in your computer

```sh
git clone https://github.com/alhashmicode/setup-reactor.git

git checkout typescript

npm install 
```

delete the main branch from your project 

```sh 
git brnach -D main
```
run webpack server 
```sh
npm run start
```
